春潮就是春天的潮汐。  
到了一定时间，海水推波助澜，迅猛上涨，达到高潮。  
过后一些时间，上涨的海水又自行退去,留下一片沙滩，出现低潮。  
如此循环重复，永不停息。海水的这种运动现象就是潮汐。 

就像我对你的爱，奔流不止，往返不息。
